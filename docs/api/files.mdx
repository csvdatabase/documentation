## Open File

Open and parse a CSDB file.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
import { openCSDB } from "@csdb/typescript";

const db = await openCSDB("payroll.csdb");
```

</TabItem>
<TabItem value="py" label="Python">

```py
from csdb import open_csdb

db = open_csdb("payroll.csdb")
```

</TabItem>
</Tabs>

JSON clients name a server-managed database in every command envelope. The server
resolves that logical identifier inside its configured data directory and opens
the file; clients cannot supply filesystem paths.

## Save File

Serialize and save a CSDB file. The TypeScript writer uses a temporary file and rename for safer replacement.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
import { saveCSDB } from "@csdb/typescript";

await saveCSDB(db, "payroll.csdb", { machineIndexes: "auto" });
```

</TabItem>
<TabItem value="py" label="Python">

```py
from csdb import save_csdb

save_csdb(db, "payroll.csdb", machine_indexes="auto")
```

</TabItem>
</Tabs>

The JSON server automatically saves each successful mutation atomically. It does
not expose a command that writes to a client-selected path.

## Parse Text

Parse text into a database.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
import { parseCSDB } from "@csdb/typescript";

const db = parseCSDB(text);
```

</TabItem>
<TabItem value="py" label="Python">

```py
from csdb import parse_csdb

db = parse_csdb(text)
```

</TabItem>
</Tabs>

## Serialize Text

Serialize a database to CSDB text.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
import { serializeCSDB } from "@csdb/typescript";

const text = serializeCSDB(db);
```

</TabItem>
<TabItem value="py" label="Python">

```py
from csdb import serialize_csdb

text = serialize_csdb(db)
```

</TabItem>
<TabItem value="json" label="JSON">

```json
{
  "database": "payroll",
  "command": { "kind": "serialize" }
}
```

The serialized CSDB document is returned as `result.text`.

</TabItem>
</Tabs>

## Configure Parsing And Serialization

Parsing supports validation control:

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
CSDBDatabase.parse(text, { validate: false });
```

</TabItem>
<TabItem value="py" label="Python">

```py
CSDBDatabase.parse(text, validate=False)
```

</TabItem>
</Tabs>

Serialization supports machine index control:

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
db.toString({ machineIndexes: "omit" });
db.toString({ machineIndexes: "auto" });
```

</TabItem>
<TabItem value="py" label="Python">

```py
db.to_string(machine_indexes="omit")
db.to_string(machine_indexes="auto")
```

</TabItem>
<TabItem value="json" label="JSON">

```json
{
  "database": "payroll",
  "command": {
    "kind": "serialize",
    "options": { "machineIndexes": "auto" }
  }
}
```

</TabItem>
</Tabs>
