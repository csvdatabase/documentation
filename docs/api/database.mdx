`CSDBDatabase` wraps a parsed document and exposes table access, SQL execution, validation, and serialization.

## Create Database Object

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
new CSDBDatabase(document: CSDBDocument)
```

</TabItem>
<TabItem value="py" label="Python">

```py
CSDBDatabase(document: CSDBDocument)
```

</TabItem>
</Tabs>

## Parse Text

Parse CSDB text into a database.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
const db = CSDBDatabase.parse(text, { validate: true });
```

</TabItem>
<TabItem value="py" label="Python">

```py
db = CSDBDatabase.parse(text, validate=True)
```

</TabItem>
</Tabs>

JSON clients select an existing server-managed database with the request envelope's
`database` field; they do not submit arbitrary files or server paths.

## Start Table Query

Start a fluent query for a table.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
const query = db.table("workers");
```

</TabItem>
<TabItem value="py" label="Python">

```py
query = db.table("workers")
```

</TabItem>
</Tabs>

JSON commands are complete, stateless plans, so there is no remote query-builder
object to create.

## Query Table

Query a table with the fluent API, SQL, or a structured JSON plan.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
const rows = db.table("workers")
  .where("email", "=", "ada@example.com")
  .select(["id"])
  .all();
```

</TabItem>
<TabItem value="py" label="Python">

```py
rows = (
    db.table("workers")
    .where("email", "=", "ada@example.com")
    .select(["id"])
    .all()
)
```

</TabItem>
<TabItem value="sql" label="SQL">

```sql
SELECT id
FROM workers
WHERE email = 'ada@example.com';
```

</TabItem>
<TabItem value="json" label="JSON">

```json
{
  "database": "payroll",
  "command": {
    "kind": "select",
    "table": "workers",
    "columns": ["id"],
    "joins": [],
    "where": {
      "type": "comparison",
      "op": "=",
      "left": { "type": "identifier", "name": "email" },
      "right": { "type": "literal", "value": "ada@example.com" }
    },
    "orderBy": [],
    "output": "objects"
  }
}
```

</TabItem>
</Tabs>

Visit [TableQuery](./table-query) for the full table query API.

## Query SQL

Run SQL text through the database API.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
const rows = db.sql(
  "SELECT id, email FROM workers WHERE email = ?",
  ["ada@example.com"]
);
```

</TabItem>
<TabItem value="py" label="Python">

```py
rows = db.sql(
    "SELECT id, email FROM workers WHERE email = ?",
    ["ada@example.com"],
)
```

</TabItem>
<TabItem value="json" label="JSON">

```json
{
  "database": "payroll",
  "command": {
    "kind": "sql",
    "statement": "SELECT id, email FROM workers WHERE email = ?",
    "params": ["ada@example.com"]
  }
}
```

</TabItem>
</Tabs>

## Create Table

Create a table from a `TableSchema`.

<Tabs groupId="language">
<TabItem value="ts" label="TypeScript">

```ts
db.createTable({
  name: "tags",
  columns: { label: "text" },
  required: ["label"],
  primary_key: { columns: ["label"] }
});
```

</TabItem>
<TabItem value="py" label="Python">

```py
db.create_table({
    "name": "tags",
    "columns": {"label": "text"},
    "required": ["label"],
    "primary_key": {"columns": ["label"]},
})
```

</TabItem>
<TabItem value="sql" label="SQL">

```sql
CREATE TABLE tags (label text primary key);
```

</TabItem>
<TabItem value="json" label="JSON">

```json
{
  "database": "payroll",
  "command": {
    "kind": "create-table",
    "schema": {
      "name": "tags",
      "columns": { "label": "text" },
      "required": ["label"],
      "primary_key": { "columns": ["label"] }
    }
  }
}
```

</TabItem>
</Tabs>
